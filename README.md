<h3>课程介绍：</h3>
<p>本套视频教程分为三大部分：基础篇、核心篇和扩展篇。</p>
<p>1. 基础篇：<a title="查看与 Scala 相关的文章" target="_blank">Scala</a>入门介绍和基本语法讲解，内容包括：Scala语言的特点和发展历史、变量和数据类型、运算符以及流程控制；</p>
